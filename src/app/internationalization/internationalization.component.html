Internationalization

The default language of Truly-UI is English(en_US) as of yet. If you want to use other languages, you can follow the instructions below.

Truly-UI provides a way to set the language of the components in the main module, so all Truly components will be translated into the configured language. To configure, simply pass the language key to the CoreModule.forRoot()
The Truly-UI Internationalization API also provides a Runtime Language Exchange service. Just use the I18nService service as an injection dependency on other services / components of your application and use the setLocale() method, passing as parameter the language
