Features

  • Allows Horizontal & Vertical Scrolling
  • Column Reordering & Resizing
  • Row Model Allow Client or Server side Pagination & Sorting
  • Intelligent Column Width Algorithms which allow you to resize the table automatically
  • Keyboard Navigation( ArrowDown, ArrownUP, Home, End, etc)
