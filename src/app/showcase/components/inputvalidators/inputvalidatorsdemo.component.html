required - Validate if the field has value.

minlength - Value field must have the length specified of characters.

pattern - Value field must match with expression (Regex) specified.

cpf - Validate Individual Registration. [Brazil]

date - Validate Date Format.

[formatDate] - dd.mm.yyyy or mm.dd.yyyy with any mask.

email - Validate Email expression.

cnpj - Validate employer code. [Brazil]

number - Field allowed only numbers.

password - Validate password field type.

[passwordRule] - Object of boolean properties to set rules - digits | uppercase | specials

The messageValidation is used on tl-input to show error messages. The box of errors messages only will appears if the input is dirty and that has validation error.

All Angular Input Events can be used too.
