Module Name: InputModule

There's a lot of ways to set up an input validator using Truly UI. Below, are some of those.

CPF
Returned Value: {{cpf}}
CNPJ
Returned Value: {{cnpj}}
DATE
Returned Value: {{date}}
NUMBER
Returned Value: {{number}}
EMAIL
Returned Value: {{email}}

cpf - Validate Individual Registration. [Brazil]

date - Validate Date Format - formatDate - dd.mm.yyyy or mm.dd.yyyy with any mask.

email - Validate Email expression.

cnpj - Validate employer code. [Brazil]

number - Field allowed only numbers.

All Angular Input Events can be used too.
